Integracja Systemów Projekt

## Użycie Lab-1
- wersja wczytująca lina po lini:
```bash
$ ./src/scripts/read_flat_file_line.py resource/katalog.txt
```
- wersja wczytująca do data frame:
```bash
$ ./src/scripts/read_flat_file_df.py resource/katalog.txt
```
